# Mental Well-Being Website – UI/UX Design

This project is a UI/UX design concept focused on creating a homepage layout for a mental well-being website. It was completed as **Task 2 of the UI/UX Design Internship at FutureInterns**.

The objective of this project was to explore how thoughtful website layouts, calming visual design, and clear content structure can contribute to a positive and supportive user experience. The design emphasizes simplicity, accessibility, and emotional comfort, which are essential for mental well-being platforms.

---

## Project Overview

The homepage design was created to simulate a real-world mental wellness website that helps users:
- Reflect on their mental state
- Build healthier habits
- Navigate content with ease and clarity

This project focuses primarily on layout planning, visual hierarchy, and user-centered design principles rather than backend functionality.

---

## Key Features

- Clean and minimal homepage layout
- Calm and balanced color palette
- Clear content hierarchy for better readability
- User-focused design approach
- Consistent spacing and alignment

---

## Tools Used

- Figma for UI design and prototyping

## Purpose of the Project

This project was created for learning and portfolio purposes to strengthen foundational UI/UX design skills, particularly in the context of website layout design and mental wellness platforms.

## Project Status

Design completed. This project currently represents a conceptual UI/UX prototype and does not include frontend or backend implementation.
